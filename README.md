# Задача Йосипа Флавія

## 1. Постановка задачі
**Об'єкт:** Коло з n людей, пронумерованих від 1 до n.  
**Ціль:** Визначити номер останнього уцілілого, якщо кожна друга людина викреслюється з кола.

**Метод розв'язання:** Метод зменшення розміру задачі на постійний множник (k=2). Ми встановлюємо співвідношення між рішенням початкової задачі та рішенням підзадачі меншого розміру.

## 2. Алгоритм та програмна реалізація
Для розв'язання використано рекурсивний підхід, що базується на структурі бінарного представлення числа n.

**Рекурсивні співвідношення:**
* **Базовий випадок:** J(1) = 1
* **Для парного n (n=2k):** J(2k)=2J(k)-1
* **Для непарного n (n=2k+1):** J(2k+1)=2J(k)+1

Код написаний мовою С у структурованому стилі. Програма підтримує роботу з файлами та аргументами командного рядка, що відповідає вимогам до "зручного інтерфейсу" та виключає необхідність прямого консольного введення даних.

## 3. Аналіз складності
Згідно з математичним аналізом рекурсивних алгоритмів:
* **Часова складність:** J(n) = \log_2 n \in \Theta(\log n).
* **Просторова складність:** O(\log n) через глибину стека рекурсії.
* **Максимальний розмір задачі:** Обмежений типом `int` та об'ємом стека викликів.
